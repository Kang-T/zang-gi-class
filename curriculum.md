# 배우기 커리큘럼 기획

완전 초보(장기를 처음 보는 초등학생)를 **정식 대국을 반칙 없이 두고 기초 전술을 아는 15급 수준**까지
데려가는 체계적 학습 코스. "읽기 → 보기 → 해보기" 흐름으로, 짧은 이야기/이론 뒤에 바로 미니게임.

## 원칙
- 저학년도 읽을 수 있게 문장은 짧고 쉽게, 이모지·SVG 도해 활용.
- 커리큘럼을 끝내면 = 15급 준비 완료 → 마지막에 "AI와 대국(15급) 도전"으로 연결.
- 콘텐츠는 앱 안에 포함(COEP 때문에 외부 이미지 불가 → 삽화는 자체 SVG 도해).

## 전체 구조 (6단원 · 약 24레슨)

| 단원 | 이름 | 내용 | 형식 |
|---|---|---|---|
| 0 | 장기와 만나기 | 장기란? · 역사(초한 楚漢) · 장기판 구경 | 이야기 + 퀴즈 |
| 1 | 장기 알 이야기 | 알의 구성(하나의 군대) · 초/한 색과 글자(卒·兵) · 기물의 값(점수) | 이야기 + 퀴즈 |
| 2 | 기물은 어떻게 움직여? | 차·포·마·상·사·졸·장 (이동+잡기) | 판 미니게임 |
| 3 | 대국의 규칙 | 차례·잡기 · 장군 · 외통(승리) · 궁성 · 빅장·반복수·한수쉼·점수 | 미니게임 + 퀴즈 |
| 4 | 기초 전술 | 공짜 말 잡기 · 말 지키기 · 장군 피하는 3가지 · 두 갈래 · 외통 1수 | 판 미니게임 |
| 5 | 첫 대국 & 졸업 | 종합 퀴즈 → 🎓 15급 수료증 → AI 대국 | 퀴즈 + 축하 |

## 레슨 형식 3가지
- **story** — 삽화 + 짧은 글 여러 장, 넘겨 보기.
- **quiz** — OX/객관식, 친절한 피드백.
- **puzzle** — `board.js` 재사용 미니게임. 목표 종류: 도착 / 잡기 / 장군 피하기 / 장군 부르기 / 외통 1수.

## 진도 & 보상
- 학습지도 UI: 단원별 레슨 노드에 완료 ✓ 표시. 진도는 localStorage 저장.
- 단원 배지: 📜 역사가 · ♟️ 알 박사 · 🎮 규칙 마스터 · ⚔️ 꼬마 전략가.
- 최종 🎓 15급 수료증 → AI 대국 연결.

## 구현 단계 (모두 완료 ✅)
- **1단계**: 프레임워크 + 학습지도 + 이야기/퀴즈 엔진 + SVG 도해 + 단원 0·1, 단원 2(퍼즐 이관).
- **2단계**: 퍼즐 엔진 일반화(장군·외통·피하기·말 지키기) + 단원 2 확장(마·상 멈춤).
- **3단계**: 단원 3(규칙 — 차례·장군·외통·궁성·빅장·반복수).
- **4단계**: 단원 4(전술 — 공짜 말·말 지키기·양수) + 단원 5(졸업 시험·15급 수료증 → AI 대국).
