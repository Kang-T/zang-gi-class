# 장기교실

초등학교 교실에서 학생들이 장기를 배우고 즐길 수 있는 웹 앱입니다.
쉬는 시간·스포츠클럽 시간에 친구와 함께, 또는 혼자서 장기를 두며 익힐 수 있도록
만들고 있어요.

> [Fairy-Stockfish](https://github.com/fairy-stockfish/Fairy-Stockfish) 엔진을 쓰는
> 개인 프로젝트(장기 훈수 도우미)를 포크해, 교실용 학습 앱으로 발전시키는 중입니다.
> 장기판·규칙·엔진 부분(`board.js` / `engine.js` / `vendor/`)은 그대로 재사용합니다.

## 실행

```bash
python3 server.py
```

브라우저에서 **http://localhost:8777** 접속. (포트 변경: `PORT=9000 python3 server.py`)

> 멀티스레드 WASM(SharedArrayBuffer)에 필요한 COOP/COEP 헤더를 서버가 붙여 주기 때문에,
> `file://` 로 직접 열지 말고 반드시 이 서버로 여세요.

## 둘이서 대국

- 두 학생이 **한 기기에서 번갈아** 판 위 돌을 눌러 둡니다. (엔진 없이 동작 → 가볍고 빠름)
- 큰 **차례 표시줄**, 각 편이 **잡은 말과 점수**(차13·포7·마5·상3·사3·병졸2) 표시
- **무르기 · 한 수 쉼 · 판 돌리기 · 다시 시작** 버튼
- 외통(장군 몰이)이 되면 **승리 축하 화면**

## AI와 대국 (급수)

- 15급 → 1급 → 초단·2단·3단 **18단계 급수 사다리**. 급수가 오를수록 엔진 강도(Skill 0~20)와
  생각 시간(0.3~2.5초)이 함께 세짐.
- **이기면 한 급수 UP, 지면 한 급수 DOWN**, 무승부는 그대로. 현재·최고 급수와 이긴 판 수는
  브라우저에 저장(localStorage)돼 다음에 이어서 도전.
- 내 편(초 먼저 / 한 나중) 선택, **무르기 · 판 돌리기 · 다시 · 항복** 버튼.
- AI 차례에는 판 클릭이 잠겨 학생이 AI 말을 대신 두지 못함.

## 배우기 (미니게임)

- 일곱 기물(차·포·마·상·사·졸·장)을 **상대 병(兵) 잡기** 한 수 퍼즐로 익힘.
- 움직일 기물이 미리 선택돼 잡을 곳(빨간 고리)이 보이고, 틀린 수는 자동으로 무르며 안내.
  2번 틀리면 정답 화살표(힌트)를 보여줌.
- 배운 기물은 브라우저에 저장(진도 ✓), 일곱 개 다 배우면 축하 화면.
- `board.js` 의 규칙 엔진(ffish)을 그대로 써서 각 기물의 실제 합법 이동만 허용.

## 온라인 대전 (실시간)

- **Supabase Realtime(Broadcast)** 로 두 브라우저가 방 코드로 붙어 수를 주고받음. HTTPS/WSS라 학교 네트워크에서도 안정적.
- 방장이 방을 만들면 4자리 코드가 나오고, 친구가 그 코드로 들어오면 자동 시작. 방장=楚 초, 도전자=漢 한.
- 자기 차례에만 판이 눌리고(차례 잠금), 항복·다시 하기·상대 이탈 처리 포함.
- 연결 설정은 `public/supabase-config.js` (anon public 키). **DB 테이블은 건드리지 않고 채널 메시지만** 사용.
- 전송부는 `online.js` 의 `tx.*` 로 감싸 두어 다른 백엔드로 교체하기 쉬움.

## 우리 반 대회 (대진표)

- 참가자 명단을 넣고 **무작위 추첨 → 단판 토너먼트 대진표** 생성. 인원이 2의 거듭제곱이 아니면 **부전승** 자동 배정.
- 승자를 누르면 다음 라운드로 진출, 최종 **🏆 우승자** 표시. 다시 누르면 정정 가능(하류 결과 자동 재계산).
- `localStorage` 저장 → 닫았다 열어도 이어짐. 스마트보드에 띄우기 좋은 가로 대진표.
- 각 대진의 **"▶ 이 대국 두기"** 로 둘이서 대국을 두 사람 이름으로 바로 열 수 있음
  (`together.html?cho=…&han=…`).

## 디자인

- 학생용 화면은 밝은 웜톤(크림 배경·흰 카드·파스텔 아이콘 타일). 규칙은 [design.md](design.md) 참고.
- 장기알은 전통 **팔각형**에 기물별 크기·글자 차등(장 크게, 졸 작게) + 안쪽 테두리·그림자.
- 선생님용 훈수 도우미는 밀도 높은 도구라 기존 다크 테마 유지.
