# 장기교실 디자인 시스템

초등학교 교실에서 쓰는 앱답게 **밝고 따뜻하고 둥근** 톤을 씁니다.
크림색 배경 위에 흰 카드, 파스텔 아이콘 타일, 부드러운 그림자 — 아이들이 보기에
편안하고 친근한 느낌을 목표로 합니다.
(레퍼런스: writingnews.vercel.app 스타일)

> **적용 범위** — 학생용 화면(홈 `index.html`, 둘이서 대국 `together.html`,
> AI와 대국 `vs-ai.html`)은 이 라이트 테마를 따릅니다.
> 선생님용 **훈수 도우미**(`advisor.html`)는 밀도 높은 분석 도구라 기존 다크 테마를 유지합니다.
>
> **구현 방식** — 라이트 테마 토큰은 `public/classroom.css` 의 `:root` 에 정의돼 있고,
> 이 파일은 `style.css` 뒤에 로드되어 학생용 화면에서만 변수를 덮어씁니다.
> 장기판(SVG)·기물 색은 `style.css` 를 공유합니다.

## 1. 색 (Color)

### 바탕·글자 (중립)
| 토큰 | 값 | 용도 |
|------|-----|------|
| `--bg` | `#FBF3E2` | 화면 배경 (따뜻한 크림) |
| `--bg-tint` | `#FFF9EE` | 살짝 밝은 영역 |
| `--surface` | `#FFFFFF` | 카드·패널 |
| `--surface-2` | `#FFF7EA` | 카드 안쪽 서브 영역 |
| `--text` | `#2E2C3A` | 본문·제목 (진한 먹색) |
| `--text-soft` | `#6E6A7C` | 보조 텍스트 |
| `--muted` | `#9A94A6` | 흐린 안내 |
| `--line` | `#EFE7D6` | 경계선 (웜 그레이) |
| `--line-strong` | `#E4D9C4` | 강한 경계선·입력칸 |

### 브랜드·의미 색
초(楚)=파랑, 한(漢)=빨강은 장기의 정체성이라 유지하되 톤을 부드럽게.
나머지 파스텔은 홈 카드·아이콘 타일에 씁니다.

| 토큰 | 값 | 파스텔(-soft) | 용도 |
|------|-----|------|------|
| `--blue` | `#2F6BE6` | `--blue-soft #DCE9FF` | 楚 초 / 주요 버튼 |
| `--blue-l` | `#5B8DEF` | | 파랑 강조 텍스트 |
| `--red` | `#EF5B4C` | `--red-soft #FFE1DC` | 漢 한 |
| `--orange` | `#FF7A2F` | `--orange-soft #FFE6D2` | 카드/강조 |
| `--purple` | `#9B5DE5` | `--purple-soft #ECE0FF` | 카드 |
| `--green` | `#23B26B` | `--green-soft #D6F3E1` | 카드/성공 |
| `--accent` | `#F5A623` | | 점수·급수 등 하이라이트(골드) |

**아이콘 타일 색 배정 (홈 모드 카드)**
- 🤝 둘이서 대국 → orange
- 🤖 AI와 대국 → blue
- 🎓 배우기 → purple
- 🌐 온라인 대전 → green
- 🏆 우리 반 대회 → red
- 🔎 훈수 도우미 → 중립(회색, 점선 테두리)

## 2. 글꼴 (Typography)

시스템 한글 글꼴을 씁니다(외부 폰트 미사용 — COEP 격리 환경에서 CDN 폰트가 막힐 수 있음).

```
font-family: "Apple SD Gothic Neo", "Malgun Gothic", "Pretendard", system-ui, sans-serif;
```

> 더 다듬고 싶으면 **Pretendard** 를 `public/vendor/` 에 자가호스팅해서 쓰면 좋습니다.

| 역할 | 크기 | 굵기 |
|------|------|------|
| 화면 제목(홈) | 32–40px | 900 |
| 섹션 제목 | 20–24px | 800 |
| 카드 이름 | 18–20px | 800 |
| 본문 | 14–16px | 500–700 |
| 보조/안내 | 12–13.5px | 500–700 |
| 큰 숫자(급수·점수) | 20–38px | 900 |

- 제목은 굵게(800~900). 아이들이 한눈에 읽도록 대비를 크게.
- 줄바꿈으로 버튼이 밀리면 탭이 빗나가므로, 고정 높이 요소는 `white-space:nowrap` 유지.

## 3. 모양 (Radius · Shadow · Spacing)

**둥근 정도** — 전반적으로 큼직하고 둥글게.
| 토큰 | 값 | 용도 |
|------|-----|------|
| `--r-sm` | `12px` | 버튼·작은 칩 |
| `--r-md` | `16px` | 입력칸·컨트롤 |
| `--r-lg` | `22px` | 카드 |
| `--r-xl` | `28px` | 큰 카드·모달 |
| 아이콘 타일 | `18px` | 파스텔 정사각 타일 |

**그림자** — 검정이 아니라 웜 브라운 기반의 아주 옅은 그림자.
| 토큰 | 값 |
|------|-----|
| `--shadow-sm` | `0 2px 6px rgba(60,50,30,.06)` |
| `--shadow-md` | `0 8px 22px rgba(60,50,30,.08)` |
| `--shadow-lg` | `0 16px 38px rgba(60,50,30,.10)` |

**여백** — 4의 배수(4/8/12/16/22/32). 카드 안쪽 패딩은 20~26px로 넉넉하게.

## 4. 컴포넌트 규칙

### 카드 (`.mode-card`, `.setup-card`, `.rank-card`, `.pinfo`)
- 배경 `--surface`, 테두리 `1px solid --line`, 반경 `--r-lg`, 그림자 `--shadow-sm`.
- 호버 시 살짝 떠오르고(`translateY(-3px)`) 그림자 `--shadow-md`, 테두리는 그 카드의 브랜드 색.
- 아이콘은 파스텔 타일(48~64px, 반경 18px) 안에 넣는다.

### 버튼 (`.cls-btn`)
- 기본: `--surface` 배경, `--line` 테두리, `--text` 글자, 반경 `--r-md`, 굵기 700.
- 주요(`.big`): `--blue` 채움 + 흰 글자 (가장 중요한 한 개 액션에만).
- 유령(`.ghost`): 투명 배경, 텍스트만.
- 누를 때 `transform: scale(.96)` 로 눌리는 느낌.

### 차례 표시줄 (`.turn-banner`)
- 초 차례 → `--blue` 채움/흰 글자, 한 차례 → `--red` 채움/흰 글자.
- 중립/종료 → 흰 카드 + `--line` 테두리.
- 장군일 때 밝기 깜빡임(`banner-flash`).

### 입력칸·셀렉트
- `--bg-tint` 또는 흰 배경, `--line-strong` 2px 테두리, 포커스 시 브랜드 색 테두리.

### 오버레이/모달
- 배경 딤: `rgba(40,34,24,.55)` (웜 톤).
- 박스: 흰 배경, 반경 `--r-xl`, 그림자 `--shadow-lg`.

---

## 5. 장기판 (그대로 유지)
- 판 바탕 `--wood #e8c887`, 격자선 `--wood-line`. 크림 배경 위에서도 자연스러움.
- 기물: 크림색 원판 + 초(파랑)/한(빨강) 테두리·글자. 내 기물은 테두리를 굵게.
- 컨테이너 그림자만 라이트 테마에 맞춰 부드럽게(`--shadow-md`).

## 6. 톤 & 보이스
- 문구는 **친근한 존댓말**, 짧고 쉬운 말. ("이기면 14급으로 올라가요!")
- 이모지를 아이콘처럼 적극 사용(🤝 🤖 🎓 🏆 🎉).
- 실패/패배도 격려로. ("아쉽게 졌어요 — 다음엔 이길 수 있어요!")
